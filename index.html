Meu primeiro post

19/03/2014 autor: Luiz Soares

Antes de tudo devo dizer que acaba de se abrir um novo horizonte de possibilidades em minha vida, até pouco tempo atrás era quase que impossível a idéia de aprender algo sobre programação, mas graças as novas diretrizes do curso, ficou muito mais interessante, pelo menos para mim, o aprendizado dessas novas linguagens, no curso Jovem Programador, estamos aprendendo a utilizar novas ferramentas que facilitam demais a vida de qualquer desenvolvedor. Conhecemos a ferramenta chamada Github, que nada mais é que uma rede social de desenvolvedores, onde se pode compartilhar tudo aquilo que se está desenvolvendo. Após conhecermos esta ferramenta nós tivemos a oportunidade de conhecer o aplicativo git que permite a ligação da sua maquina local com o servidor do Github, nos aprofundamos um pouco nesse aplicativo e aprendemos alguns comandos como, git status, que permite saber se há arquivos e se estamos conectados com o servidor, existe também o comando git log, que nos permite visualizar todas modificações feitas em nossa pasta, git add “nome do arquivo”, permite adicionar um novo arquivo a nossa pasta no servidor, git commit –m “mensagem” deixa uma mensagem no servidor no arquivo que se modificou para saber quem e como se modificou, git push que sobe para o servidor as modificações realizadas no arquivo, até agora foram estas as novas ferramentas que conhecemos, esperamos conhecer cada vez mais.
